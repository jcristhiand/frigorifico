//English
This repository contains a prototype application for a cold storage company.

The app allows users to view:
- Product storage areas.
- A catalog of available products.
- A descriptive profile of the company.
- Required storage conditions.
- Labels associated with each product.
- Available inventory quantities.

This prototype was developed as part of an academic project for the faculty, with the goal of applying software design and development concepts in a real-world context.




//Spanish
Este repositorio contiene el prototipo de una aplicación para una empresa frigorífica.

La app permite visualizar:
- Las zonas de conservación de los productos.
- El catálogo de productos disponibles.
- Una monografía descriptiva de la empresa.
- Las condiciones de conservación requeridas.
- Las etiquetas asociadas a cada producto.
- Las cantidades disponibles en inventario.
- 
Este prototipo fue desarrollado como parte de un proyecto académico de la facultad, con el objetivo de aplicar conceptos de diseño y desarrollo de software en un contexto real.
